# 🎓 Student Management System

### *A Console-Based Java Application*

![Java](https://img.shields.io/badge/Java-ED8B00?style=for-the-badge&logo=openjdk&logoColor=white)
![OOP](https://img.shields.io/badge/OOP-Concepts-blue?style=for-the-badge)

<br>

*A console-based Java project developed to manage student records using **Core Java**, **Object-Oriented Programming (OOP)**, and **ArrayList**.*
*This project performs complete **CRUD operations** (Create, Read, Update, Delete) through a menu-driven interface.*

---

</div>

## ✨ Features

| # | Feature | Description |
|---|---------|-------------|
| 1 | ➕ **Add Student** | Register a new student record |
| 2 | 📋 **View All Students** | Display all stored student records |
| 3 | 🔍 **Search Student by Roll Number** | Find a specific student quickly |
| 4 | ✏️ **Update Student Details** | Modify existing student information |
| 5 | 🗑️ **Delete Student Record** | Remove a student from the system |
| 6 | 🚪 **Exit Application** | Safely close the program |

## ⚙️ How It Works

The program stores student records in an `ArrayList<Student>` during runtime.

Each student contains:

```java
🔢  Roll Number
👤  Name
🎂  Age
📚  Course
```

> The system allows users to perform operations using a **menu-driven console**.

---

## 🚀 How to Run

**Step 1 — Compile**

```bash
javac Student.java StudentManagementSystem.java
```

**Step 2 — Execute**

```bash
java StudentManagementSystem
```
